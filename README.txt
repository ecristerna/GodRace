Instrucciones de uso:

- Ejecute la clase GodRace.java para iniciar el juego.
- Utilice las teclas W-A-S-D para navegar por el menú.
- Utilice la tecla ENTER para seleccionar una opción.
- Utiliza la tecla ENTER para volver al menú, estando en la pantalla de instrucciones.
- Estando en la pantalla de selección de personaje, siendo el jugador 1 utilice las teclas W-A-S-D para navegar por los personajes y la tecla F para seleccionarlo; siendo el jugador 2, utilice las flechas para navegar por los personajes y la tecla N para seleccionarlo.
- Estando en la pantalla de selección de pista, utilice las teclas W-A-S-D para navegar por las opciones y la tecla ENTER para seleccionarla.
- Una vez en el juego, utilice las teclas W-A-S-D para mover al personaje 1 a través de la pista; para mover al personaje 2, utilice las flechas del teclado.
- Para reiniciar el juego, presione la tecla ENTER estando en la pantalla de Game Over.


Lo que funciona:

- Menú principal y selección de opciones.
- Navegación y selección de personajes para ambos jugadores.
- Navegación y selección de pista para juego.
- Choque y reducción de vida con los obstáculos.
- Uso de barra de vida.
- Reinicio de juego al terminar.

Lo que falta:

- Utilización de powerups dependiendo de los personajes.
- Animación de los personajes.
- Imagen de créditos al finalizar.
- Imagen del personaje ganador en la pantalla de Game Over.
